Installation

Install Stratifyd SDK

####Requirements

Installing Stratifyd SDK is quite easy. However, you do need to have a couple of other things installed first:

  • Node.js
  • Git

Installation

If your computer already has these, congratulations! Just install Stratifyd SDK with npm:

Connect to Straitfyd Portal

Once you have installed SDK, you need to get your credentials to access Straityfd serivices.

  1. Go to the main Stratifyd Portal for your enterprise, usually it is https://your-subdomain.stratifyd.com or https://your-subdomain.stratifyd.cn
  2. Login with your credentials, in the home page, go to seetings view and click on Create API Key
  3. Download you API Key as a json file. The API key has the same user groups as the owner.

Warning! It’s your resposibility to protect the credential.
A json with a complete credential will be pasted to your clipboard.

NodeJS has a easy way to load json.

If you want to use the command line, save the credential to .signalsrc under your working folder.

####Getting Started
Import your API Key into your NodeJS code.

Replace ‘en’ with ‘zh’ if you are using china node

Install Signals Command Line Tool

Command line tool is no longer included in the signals-api, use signals-cli instead

You are now connected to Straitfyd, please review other chapters to see how to run analysis, upload data and create groups.
