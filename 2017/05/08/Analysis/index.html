#Analysis

Create a new analysis

You need a CSV/Excel/Json data and an analysis configuration to create an analysis.
The job configuration tells Stratifyd how to process the data in the file and provides initial meta data for the analysis.
Check out more detailed analysis configuration/parameters that you can use in the "Analysis Configuration" section.

Command Line

List running analysis

Get analysis meta

You need a job id to locate a specific job.
Job ids can be found in the response of creating a new job or the previous job status calls.

Analysis Configurations

Job configuration tells Signals how to process the data.
The most important part is the params which define the data type of each column in the csv file.
You have to assign an array of column names to each index.
